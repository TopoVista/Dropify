STEP 2 — Create Virtual Environment
===================================

Run:

`python -m venv venv`

This creates:

`my-dropify-backend/└── venv/`

* * *

STEP 3 — Activate venv (IMPORTANT)
==================================

Since you're on Windows PowerShell:

`venv\Scripts\activate`

If it fails with execution policy error, run:

`Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`

Then activate again.

* * *

### ✅ When activated correctly, your terminal should show:

`(venv) PS C:\Users\...\my-dropify-backend>`

If you do NOT see `(venv)`, you are not inside it.
